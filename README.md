# WildWatch PK | AI-Powered Wildlife Extinction Risk Analyzer for Pakistan

## Overview

WildWatch PK is an end-to-end Machine Learning project designed to analyze wildlife population trends and identify species that may be at risk of extinction in Pakistan.

The project was inspired by a real-world conservation challenge: despite being home to unique and ecologically important wildlife, Pakistan lacks accessible data-driven systems that can continuously monitor species populations and provide early warnings before biodiversity loss becomes irreversible.

This project demonstrates how Artificial Intelligence and Data Science can be applied to environmental conservation by transforming raw wildlife data into meaningful insights, risk assessments, and interactive visualizations.

---

## Project Objectives

* Analyze wildlife population and biological data.
* Identify species showing signs of population decline.
* Predict extinction risk levels using Machine Learning.
* Visualize conservation insights through an interactive dashboard.
* Demonstrate the practical application of AI in environmental protection.

---

## Project Workflow

### 1. Data Collection

Wildlife data was collected and compiled from publicly available biodiversity resources and conservation references.

### 2. Data Preprocessing

* Missing value handling
* Data cleaning
* Feature transformation
* Label encoding
* Outlier analysis
* Feature scaling and normalization

### 3. Exploratory Data Analysis

* Population distribution analysis
* Correlation analysis
* Risk pattern discovery
* Statistical visualization

### 4. Feature Engineering

Relevant biological and population-related attributes were selected and transformed to improve model performance and prediction quality.

### 5. Machine Learning Modeling

Multiple machine learning techniques were explored to classify species into extinction risk categories:

* Low Risk
* Medium Risk
* High Risk

### 6. Model Evaluation

Models were evaluated using standard classification metrics and prediction analysis to assess performance and reliability.

### 7. Dashboard Development

An interactive Streamlit dashboard was developed to present predictions and wildlife insights in a user-friendly format suitable for researchers, students, and conservation enthusiasts.

## Why WildWatch PK?

WildWatch PK was developed to explore how Machine Learning can contribute to wildlife conservation and biodiversity protection. While this project was created as part of a learning journey in Artificial Intelligence, it was built around a real environmental problem that affects ecosystems across Pakistan.

The goal is not only to predict extinction risk but also to demonstrate how data-driven decision making can support conservation efforts and encourage the use of AI for meaningful social and environmental impact.
